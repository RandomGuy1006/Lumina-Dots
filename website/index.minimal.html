Features

What Makes Lumina Different

A dotfiles framework built for real hardware, not just screenshots. Every feature is tested on a Lenovo LOQ 15IRX9 with dual-GPU NVIDIA.

Spring-Physics Animations

Android/iOS-like overshoot curves give windows a natural bounce without being distracting.

Material You Theming

Matugen generates 13 token files from your wallpaper. Every app, bar, lock screen, and terminal updates in sync.

Btrfs Snapshots

Pre-install snapshot with rollback support. One command away from a working system.

NVIDIA-Ready

Integrated, hybrid, or NVIDIA-only mode switching via env vars. Sleep works. Games run.

Tested & Validated

CI runs static analysis, doc validation, theme rendering, and integration smoke tests on every commit.

Host Profiles

Hardware-specific configs for LOQ 15IRX9 with a generic fallback. Add your own host in minutes.

Stack

The Toolchain

Each component was chosen for stability, performance, and Wayland-first design.

Hyprland

Dynamic tiling Wayland compositor with spring-physics animations and per-monitor workspaces.

Zsh + p10k

Powerlevel10k prompt with instant prompt, syntax highlighting, autosuggestions, and fzf.

Matugen

Material You color extraction from wallpaper. Generates 13 token files across the stack.

HyprPanel

GTK status bar with workspaces, volume, brightness, network, Bluetooth, and theme controls.

Hyprlock

GPU-accelerated lock screen with blurred wallpaper, clock, and password input.

Hypridle

Multi-stage idle: dim, lock, display off, suspend on battery.

swww

Animated wallpaper transitions with smooth fade effects on theme change.

Walker

GTK launcher for app search, clipboard history, calculator, and emoji picker.

Ghostty

GPU-accelerated terminal with Material You themed color scheme.

Doctor

Built-in diagnostics for services, packages, battery health, and config drift.

uwsm

Universal Wayland session manager for environment isolation and systemd integration.

Btrfs + Snapper

Automatic pre-install snapshots with manual rollback for safe experimentation.

Install

Installation Guide

Fresh Arch install recommended. The script handles partitioning, packages, dotfiles, and Btrfs snapshots.

Guided Install (archinstall) — Recommended
0

BIOS prep

Secure Boot off, UEFI, AHCI. USB flash:

# dd if=archlinux-*.iso of=/dev/sdX bs=4M status=progress && sync
1

Wi-Fi

iwctl
device list
station wlan0 scan
station wlan0 get-networks
station wlan0 connect "YourNetwork"
exit
2

Disk + Btrfs subvolumes

gdisk /dev/nvme0n1
mkfs.fat -F32 /dev/nvme0n1p1
mkfs.btrfs -f /dev/nvme0n1p2
mount /dev/nvme0n1p2 /mnt
btrfs subvolume create /mnt/@
btrfs subvolume create /mnt/@home
btrfs subvolume create /mnt/@snapshots
btrfs subvolume create /mnt/@var_log
btrfs subvolume create /mnt/@var_cache_pacman
umount /mnt
mount -o noatime,compress=zstd:3,subvol=@ /dev/nvme0n1p2 /mnt
mkdir -p /mnt/{boot,home,.snapshots,var/log,var/cache/pacman/pkg}
mount -o subvol=@home /dev/nvme0n1p2 /mnt/home
mount -o subvol=@snapshots /dev/nvme0n1p2 /mnt/.snapshots
mount /dev/nvme0n1p1 /mnt/boot
3

archinstall --advanced

# profile: minimal
# bootloader: systemd-boot or rEFInd
# audio: pipewire
# network: NetworkManager
# additional packages: git sudo neovim networkmanager
4

First boot

git clone https://github.com/RandomGuy1006/lumina-dots ~/lumina-dots
cd ~/lumina-dots && bash install.sh install --host=loq-15irx9
5

Post-install

dotfiles doctor
dotfiles theme ~/Pictures/Wallpapers/image.jpg
Manual Install
1

Base system

pacstrap -K /mnt base linux linux-lts linux-firmware intel-ucode \
  sudo git zsh neovim networkmanager btrfs-progs snapper snap-pac
2

fstab + chroot

genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt
3

User + services

useradd -m -G wheel,audio,video,input,network arch
systemctl enable NetworkManager
4

Clone lumina

git clone https://github.com/RandomGuy1006/lumina-dots /home/arch/lumina-dots
chown -R arch:arch /home/arch/lumina-dots
5

Run installer

su - arch
cd ~/lumina-dots
bash install.sh install --host=loq-15irx9
# modes: dotfiles-only / packages-only / hardware-only

Experience

Why Lumina feels premium on Arch

Fast Session Bring-up

UWSM + systemd user targets ensure deterministic startup ordering for swww, panel, idle, and lock.

Wallpaper-native Theming

One wallpaper drives Matugen colors through Hyprland, Hyprlock, Ghostty, GTK, Walker, Wlogout, and Hyprpanel.

Built-in Recovery Paths

Strict snapshot gates + rollback flows make updates safer. On GRUB, snapshots appear directly in the boot menu via grub-btrfs.

Unified Visual Tokens

Spacing on a 4/8/12/16/24 grid. Token rendering aborts on missing keys to prevent malformed configs.

OCR + Optional Local AI

Super+Shift+O OCRs a region, logs history, and can refine via a local hook or Ollama. Falls back to raw OCR on failure.

Host Profiles

--host=loq-15irx9 applies NVIDIA, i915, logind, and monitor tuning; generic skips machine-specific overlays.

Workspace Templates

TOML templates launch dev/media layouts — apply with dotfiles workspace-template or Hypr binds.

Recovery

When Things Go Wrong

Every install and update creates a Btrfs snapshot. Recovery is a single command.

Rollback

rollback.sh lists snapshots by date and restores the selected one. Works from any TTY.

Doctor

doctor.sh checks 30+ diagnostics — services, packages, config drift, battery health.

Recovery Guide

See docs/recovery.md for step-by-step TTY recovery, chroot, and manual rollback.
