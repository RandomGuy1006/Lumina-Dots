# Pomodoro

Pomodoro is a lightweight Phase 6 timer.

- App: `apps/pomodoro/lumina-pomodoro.py`
- CLI: `lumina-pomodoro`
- Dotfiles command: `dotfiles pomodoro`
- Keybind: `Super + Shift + T`
- State: `pomodoro.json` under Lumina Core state paths

It works without AI, network, or a background service.
