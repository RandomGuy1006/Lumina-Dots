# Lumina Hub

Lumina Hub is the Phase 4 central product surface for first-party Lumina tools.

- App: `apps/lumina-hub/lumina-hub.py`
- CLI: `lumina-hub`
- Dotfiles command: `dotfiles hub`
- Keybind: `Super + L`

Hub shows the active Lumina mode, Lumina Shell availability, quick links to first-party apps, and recent activity from `activity/history.json`.

Keybind tradeoff: `Super + L` previously focused right in the Vim-style focus set. Right focus remains available through `Super + Right` and `Super + Alt + L`.
