# Feature Status

## Production

Installation, host profiles, UWSM session startup, HyprPanel fallback, Walker, Hyprlock, theme generation, wallpaper switching, doctor, validation, backups, offline Btrfs rollback, Lumina popup/OSD/modes, control center, keybind overlay, and recovery shell.

## Optional

Lumina AI, NVIDIA offload, Waypaper, EasyEffects, Pomodoro, cleanup manager, workspace templates, scratch tools, presentation mode, focus mode, media overlay, and color picker.

## Experimental

The live Hyprland event stream and external contract consumers are stable enough for testing but do not replace production GTK/HyprPanel surfaces.

## Planned

Theme personalities, automatic multi-host selection, benchmark evidence, broader storage health checks, and a Quickshell proof of concept after contract-readiness acceptance.

Every production or optional feature must document dependencies, failure behavior, state, recovery, and verification. Experimental features must remain removable without affecting login or recovery.
