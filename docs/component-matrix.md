# Component matrix

| Layer | Choice | Why it stays | Notes |
|---|---|---|---|
| Compositor | Hyprland | Best motion/control blend for Wayland power users | Config kept modular |
| Session manager | UWSM | Cleaner environment and lifecycle than launching everything from Hyprland alone | `.zprofile` starts it from tty1 |
| Login path | getty autologin + Hyprlock | No display manager tax, still keeps a locked entry point | Immediate lock on session start |
| Shell feedback | Lumina Shell + HyprPanel fallback | First-party OSD, modes, and popups without losing the stable bar/widget fallback | Lumina Shell state under `~/.local/state/lumina`; HyprPanel runtime config generated from base + theme |
| Launcher | Walker | Modern, visually adaptable, keyboard-first | Custom glass CSS |
| Lockscreen | Hyprlock | Native, elegant, themeable | Color vars generated by Matugen |
| Idle manager | Hypridle | Native and predictable | Suspends by default and only escalates to suspend-then-hibernate when resume is configured |
| Wallpaper | swww + Waypaper | Great transitions plus easy GUI selection | `wallpaper-apply` is the source of truth |
| Adaptive colors | Matugen | Strong results with low maintenance overhead | Drives multiple outputs |
| GTK | adw-gtk-theme + nwg-look | Good default app coherence without heavy theme hacks | Static shell theme, dynamic accents elsewhere |
| Icons | Papirus-Dark | Mature and consistent | Low maintenance |
| Cursor | Bibata-Modern-Ice | Best visual fit with the rest of the system | Set in UWSM env and GTK |
| Terminal | Ghostty | Typography and polish win | Dynamic theme file |
| Shell | Zsh + Starship | Fast, readable, low-friction | Minimal modular setup |
| Editor | LazyVim | Strong defaults with lower upkeep than custom Neovim | Bootstrap only; easy to grow |
| File managers | Nautilus + Yazi | GUI convenience plus terminal speed | Covers both workflows well |
| Audio | PipeWire + WirePlumber + EasyEffects | Best modern Linux audio baseline | No extra notification daemon needed |
| Bluetooth | Bluez + Blueman | Standard, reliable | Fits HyprPanel menus well |
| Power / thermals | auto-cpufreq + UPower + thermald | Better LOQ battery and thermal behavior without a giant control stack | Intel-first baseline |
| Clipboard | Cliphist + wl-clipboard | Lightweight, reliable | systemd user watchers |
| Screenshot | Grimblast + Satty | Native capture with optional annotation | Keybinds stay simple |
| Power menu | Wlogout | Clean and themeable | Styled to match the rest |
| Recovery | Btrfs + Snapper + snap-pac + linux-lts | Production-grade rollback story | Integrated into update flow |
