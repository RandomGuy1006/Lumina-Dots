# Default profile

This repo is designed around an Intel-first Lenovo LOQ 15IRX9 profile:

- systemd-boot
- Btrfs subvolumes with Snapper and snap-pac
- TTY autologin with UWSM
- Hyprlock shown immediately after session start
- Matugen-driven theming from the active wallpaper
- HyprPanel, Walker, Ghostty, and Hyprland all sharing one color system
